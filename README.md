# Weather-app-qt

 It is made using OpenWeatherMap API. 
 Any location can be entered to get the weather details like-
 
    • Temperature(1] Feels like 2] MaxTemp 3] MinTemp)
    • Humidity
    • Visibility
    • Pressure
    • Sunrise
    • Sunset
    • WindDirection and speed
    
Language used
    • C++ (Object Oriented)
    
Tool
    • Qt Creator
